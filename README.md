# Apriori-Optimise-IA-JobMarket
Implémentation d’Apriori avec support minimal automatisé basé sur une formule hybride (moyenne + écart-type). Comparaison détaillée avec l’Apriori classique (Weka) sur des données du marché de l’emploi en IA. Visualisations, scripts d’analyse, export Python → ARFF, et métriques de règles d’association.
